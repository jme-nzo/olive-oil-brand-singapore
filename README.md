# Olive Oil Brand Singapore

A cinematic, single-page e-commerce prototype for a fictional Singapore-based olive oil estate. Built with Vite + React, Tailwind CSS, GSAP, and Lucide React.

## Sections
- **Navbar** with pill-shaped container and Sign In modal (blurred backdrop, cinematic ease).
- **Hero** — split layout, Gambarino display type, slow GSAP scale animation on the hero image.
- **Shop Grid** — 3 cold-pressed products, hover reveals Quick Add.
- **Farm Story** — parallax narrative section with scroll-driven GSAP animations.
- **Process** — three-step grove-to-bottle storytelling.
- **Footer** — Rich Soil background, newsletter, locally-harvested badge.

## Design System
- **Olive Canopy** `#4A5D23` · **Liquid Gold** `#D4AF37` · **Warm Linen** `#FDFBF7` · **Crushed Stone** `#F4F1EA` · **Rich Soil** `#2C2A25`
- Display: Gambarino · Body: Inter · Mono: Space Mono

## Run
```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # production build → dist/
```

## Deploy
Static Vite output. Drop the `dist/` folder onto Vercel/Netlify/Cloudflare Pages, or import the GitHub repo directly into Vercel for automatic builds.
